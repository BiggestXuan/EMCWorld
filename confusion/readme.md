## 范围？
- 仅对 CraftTweaker 的脚本生效。
## 原始代码呢？

## 我能的包能用这个吗？
- 我不确定，这是针对 EMCWorld 用的，如果要用需要满足以下条件：
  
1.所有脚本只有一个main方法作为入口。
  
2.所有function必须声明优先级。
  
3.所有function必须以}结尾。
  
4.可能还有其他问题，但如果要自己用出问题了别找我。
